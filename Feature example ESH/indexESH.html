The HUD

Click on any element in the Live Design Surface will bring up the element HUD

The HUd will have the following

  • The Tag Name
  • The classes and IDs in case any 
  • Option to add more clasees /ID

On selecting multiple elements the HUD will auto update just like the Tag selector does - i.e. Draw a HUD on top of the parent element 

Classes and IDs

Clicking on the + icon will let one add classes and IDs into the HTML

Apart from adding classes and IDs

  • Hinting for classes and IDs
    • . for list of all classes
    • # for list of all IDs
    • Classes added by default
  • Add multiple classes
  • Classes are added by default unless user specifies an #

Other Features 

Apart from the obvious features the Editable Selectors HUD also 

  • Maintained in Sync with all Live tools
  • Works seemlessly with CSS Designer
  • Supports Live UNdo and Redo ops
